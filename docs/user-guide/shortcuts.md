# Shortcuts

Open the Command Palette with Ctrl/Command+Shift+P. Common defaults include Ctrl/Command+N/O/S, Undo/Redo, P Pencil, E Eraser, I Eyedropper, G Fill, L Line, R Rectangle, S Selection, M Move, brackets for adjacent frames, and Enter for playback.

The Keybinding Editor imports and exports versioned JSON. Chords are normalized by platform. Exact same-context conflicts require replacement; unsafe unmodified characters, text-input conflicts, and OS-reserved shortcuts are warned. Commands do not fire while typing in an input.
