# Getting started

Suwol Pixel Studio 0.6 is a desktop pixel editor preparing the stable 1.0 release. Create an RGBA or indexed document, choose a Pixel Layer, select a tool, and save as `.suwolpixel`. The native menu and Command Palette expose the same command registry. Unsaved changes are marked with a dot on the document tab.

Use System, Dark, or Light theme and 80–200% UI scale from the status bar. Korean and English are supported and persist across restarts. Start with a small canvas, save early, and keep recovery enabled by leaving the app's user-data directory writable.
