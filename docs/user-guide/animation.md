# Animation

Frames have stable IDs and millisecond durations. A Cel connects one layer and frame to pixel or tilemap data. Linked Cels share image storage; editing one changes every link, while Unlink clones the selected Cel.

Playback supports Loop, Once, and Ping-pong without changing history. Tags define named frame ranges and playback direction. Onion Skin can use the active layer or full composite. Export supports PNG sequence, sprite sheet plus JSON, GIF, and APNG; GIF cannot preserve partial alpha.
