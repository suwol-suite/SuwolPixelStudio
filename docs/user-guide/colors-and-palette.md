# Colors and Palette

Suwol Pixel Studio keeps the color you are drawing with, your recent color history, and the document palette separate.

## Foreground and background colors

The foreground color is used by Pencil, Fill, Line, Rectangle, and Ellipse. Left-click a document-palette swatch to set it. The background color is a secondary color; right-click a swatch to set it. Use the swap button to exchange only these two colors.

Both colors appear as overlapping swatches in the Tool Options Bar and at the top of the Palette panel. Click either swatch to open the system color picker. You can also enter a HEX value and alpha value in the Palette panel.

## Recent colors

Recent Colors is a usage history. A color is recorded after you confirm it or use it to draw. Clicking a recent color selects it without moving the list. Duplicate entries are removed automatically, and the history is stored in app settings rather than in the document undo history.

## Document Palette

Document Palette belongs to the current document. Use **Add Current Color** to add the foreground color. Left-click sets the foreground color, right-click sets the background color, and double-click opens the selected-color editor. Drag swatches to reorder them.

Palette order changes only when you explicitly reorder or sort it. In Indexed Color documents, slot indices define pixel values, so selecting a color never reorders the palette. Reordering remaps indexed pixels to preserve the visible image.
