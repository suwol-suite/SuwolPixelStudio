# Desktop assets

이 디렉터리는 Electron Main/Forge가 소유하는 정적 데스크톱 자산 경계다. `icon-source.png`가 배포 아이콘의 원본이며 `icon.icns`, `icon.ico`, `linux/studio.suwol.pixel.png`는 각각 macOS, Windows, Linux ZIP/AppImage와 About 화면에 사용한다. `linux/application-x-suwol-pixel-studio.png`는 같은 디자인의 MIME 문서 아이콘이다. M6는 macOS hardened-runtime entitlement와 Linux desktop/MIME metadata도 함께 포함한다.
